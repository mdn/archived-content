O Firefox 3 oferece algumas melhorias à sua implementação CSS. Enquanto estas mudanças são documentadas nas páginas apropriadas da documentação, este artigo proporciona uma introdução destas melhorias para que desenvolvedores possam aprender mais facilmente quais foram estas mudanças.

  • Os valores inline-block e inline-table da propriedade {{ Cssxref("display") }} estão agora implementados.
  • A propriedade {{ Cssxref("font-size-adjust") }} agora trabalha em todas as plataformas; anteriormente ela era suportada somente no Windows.
  • Suporte de rgba() e hsla() para {{ Cssxref("color") }} ({{ Bug(147017) }})
  • Suporte à pseudo-classe {{ Cssxref(":default") }} ({{ Bug(302186) }})
  • Os valores -moz-max-content, -moz-min-content, -moz-fit-content e -moz-available foram adicionados a {{ Cssxref("width") }}, {{ Cssxref("min-width") }} e {{ Cssxref("max-width") }} ({{ Bug(311415) }} e {{ Bug(402706) }})
  • Hífens leves em HTML (&shy;) são agora suportados.
  • Abas pré-formatadas trabalham muito melhor em fontes proporcionais, seguindo as especificações do CSS 2.1.
  • A propriedade {{ Cssxref("ime-mode") }} é suportada.
  • A propriedade CSS text-rendering para HTML é agora suportada ({{ Bug(387969) }}).
  • As propriedades CSS -moz-border-*-start e -moz-border-*-end foram implementadas ({{ Bug(74880) }}).
  • -moz-initial está agora implementado para quase todas as propriedades CSS (exceto quotes e -moz-border-*-colors) {{ Bug(80887) }}.
  • window.getComputedStyle() agora suporta todas as propriedades CSS suportadas ({{ Bug(316981) }}).
  • O valor none da propriedade {{ Cssxref("content") }} é agora suportado ({{ Bug(378535) }}).
  • O valor none da propriedade {{ Cssxref("cursor") }} é agora suportado ({{ Bug(346690) }}).
  • Imagens de fundo agora prendem-se adequadamente a -moz-border-radius ({{ Bug(24998) }}).
  • O valor pre-wrap da propriedade {{ Cssxref("white-space") }} é agora suportado ({{ Bug(261081) }}).
  • Seletores como :first-child, :only-child, :last-child ({{ Bug(73586) }}), :empty ({{ Bug(98997) }}), ou o combinador ({{ Bug(229915) }}) são agora atualizados dinamicamente ({{ Bug(401291) }}).
  • Valores negativos da propriedade {{ Cssxref("z-index") }} são suportados normalmente.
