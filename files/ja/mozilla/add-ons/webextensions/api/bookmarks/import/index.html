ブックマークを HTML ブックマークファイルからインポートします。

構文

browser.bookmarks.import(
  function() {...} // 関数（省略可）
)

引数

callbackOptional
function.

ブラウザ実装状況

現在、互換性データを可読形式の JSON フォーマットに置き換えているところです。 この互換性一覧は古い形式を使っており、これに含まれるデータの置き換えが済んでいません。 手助けしていただける場合は、こちらから！
