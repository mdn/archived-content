非标准
该特性是非标准的，请尽量不要在生产环境中使用它！

概述

将字符串中包含的特殊字符进行转义(反斜杠),然后在字符串两边各加上一个双引号(")并返回,并不修改原字符串.

语法

str.quote()

示例

str str.quote() eval(str.quote())
Hello world! "Hello world!" Hello world!
Hello
        world!		 "Hello\n\tworld!" Hello
        world!
" \ — ' "\" \\ \u2014 '" " \ — '

规范

不在任何规范中。实现于 JavaScript 1.3.
