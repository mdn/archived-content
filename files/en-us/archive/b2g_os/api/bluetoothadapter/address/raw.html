This API is available on Firefox OS for internal applications only.

Summary

The address property value provides the address of the device's adapter on the Bluetooth micro-network.

Syntax

var address = instanceOfBluetoothAdapter.address

Value

Return a string, which should be formatted as xx:xx:xx:xx:xx:xx. x could be [0-9A-F].

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
