This API is available on Firefox OS for internal applications only.

Summary

The setDiscoverable method is used to change the value of the discoverable property for the device's adapter.

Syntax

var request = instanceOfBluetoothAdapter.setDiscoverable(flag);

Parameters

flag
A boolean indicating if the device is discoverable (true) or not (false).

Returns

A DOMRequest object to handle the success or error of the operation. In case of success, if the value has been set to true, the device remains discoverable for the duration of the discoverableTimeout property.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
