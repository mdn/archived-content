    1. FMRadio

This API is available on Firefox or Firefox OS for installed or higher privileged applications.

Summary

The frequencyLowerBound property value indicates the minimum frequency down to which the seek method searches for radio stations.

Syntax

var max = instanceOfFMRadio.frequencyLowerBound

Value

Return a number representing a frequence expressed in MHz (MegaHertz).

Specification

Not part of any specification.
