Summary

The status property indicates if the connection is active (true) or not (false).

Syntax

var status = instanceOfBluetoothStatusChangedEvent.status

Value

Return a boolean.

Specification

Not part of any specification yet. It should be discussed as part of the W3C's System Applications Working Group.
