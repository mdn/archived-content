This API is available on Firefox OS for internal applications only.

The ondisconnecting event handler of the TelephonyCall interface is a reference to a function to be called to handle the Telephony API's disconnecting event.

This event occurs when the call starts disconnecting from the phone network. The event handler is called with a CallEvent as its only parameter.

Syntax

myCall.ondisconnecting = function() { ... };

Examples

// Telephony object
var tel = navigator.mozTelephony;

// Place a call
tel.dial("123456789").then(function(call) {
  call.number;

  call.ondisconnecting = function(e) {
    console.log('Your call to ' + e.call.id + ' is hanging up.');
  }
});
