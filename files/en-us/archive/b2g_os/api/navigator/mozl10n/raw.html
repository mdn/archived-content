{{APIRef("L10N API")}}{{Non-standard_Header}}

navigator.mozL10n returns an {{DOMxRef("L10n")}} object you can use to access localization resources and respond to localization events.

Syntax

var l10n = navigator.mozL10n;

Value

navigator.mozL10n returns an {{DOMxRef("L10n")}} object.

Specifications

Not part of any specification; however, note that this API will be replaced by L20n in the future.
