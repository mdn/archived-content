Sumário

Retorna um objeto {{ domxref("BluetoothManager") }} que você pode usar para obter um objeto BluetoothAdapter.

Sintaxe

var bluetooth = window.navigator.mozBluetooth;

Valor

navigator.mozBluetooth é um objeto {{domxref("BluetoothManager")}}.

Especificação

Não é parte de nenhuma especificação.
