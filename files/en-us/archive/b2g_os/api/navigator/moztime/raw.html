Summary

Returns a {{ domxref("MozTimeManager") }} object you can use to change the device's time.

Syntax

var time = window.navigator.mozTime;

Value

navigator.moztime is a {{domxref("MozTimeManager")}} object.

Specifications

Not part of any specification.
