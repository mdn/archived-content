This API is available on Firefox OS for internal applications only.

Summary

The closed property indicates whether or not the settings lock is closed.

Syntax

var isClosed = instanceOfSettingsLock.closed;

Value

Returns a Boolean indicating the state of the settings lock; if true, the lock is no longer active. If false, the lock is still the active lock.

Specification

Not part of any specification yet; however, this API will be discussed at W3C as part of the System Applications Working Group.
