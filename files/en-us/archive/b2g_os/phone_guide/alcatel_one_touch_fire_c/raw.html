The Alcatel One Touch Fire C smartphone is powered by Firefox OS; it has a 1.2 GHz Dual Core processor and 2 MP Rear Camera and is available in India.

Get a device

  • Currently Not Available

Countries Available

  • India
  • Uruguay

Updating & upgrading the software

The device's system image is locked, so you can only update the software when the network operator releases an OTA update. The device system image is based on Firefox OS 1.2.

Device specifications

You can find more of the device specifications listed on Alcatel's Website under Specification tab.

  • CPU: Dual Core 1.2 GHz CPU
  • Camera: 2 MP Rear
  • Battery: 1300mAh
  • Display: 3.5” HVGA, TFT 262K color display, Capacitive Screen with gesture.
  • Resolution: 320x480 pixels
  • Internal memory: 4GB ROM + 512MB RAM
  • External memory: MicroSD Card: up to 32GB
  • Size: 112.2 x 62 x 11.95mm
  • Weight: Approx.100g
