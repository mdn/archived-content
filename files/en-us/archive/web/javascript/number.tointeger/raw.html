The Number.toInteger() method used to evaluate the passed value and convert it to an integer, but its implementation has been removed.

If the target value is {{jsxref("NaN")}}, {{jsxref("null")}} or {{jsxref("undefined")}}, 0 is returned. If the target value is false, 0 is returned and if true, 1 is returned.

Syntax

Number.toInteger(number)

Parameters

number
The value to be converted to an integer.

Examples

Using toInteger

Number.toInteger(0.1);     // 0
Number.toInteger(1);       // 1
Number.toInteger(Math.PI); // 3
Number.toInteger(null);    // 0

Specifications

Not part of any standard. Number.toInteger() was part of the draft ECMAScript 6 specification, but has been removed on August 23, 2013 in Draft Rev 17.

Browser compatibility

Not supported anywhere. Historically supported in Firefox 16 till 31.
