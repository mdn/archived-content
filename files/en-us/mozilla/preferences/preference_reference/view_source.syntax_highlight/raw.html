The preference view_source.syntax_highlight controls whether markup in the view source view is syntax highlighted.

  • Type:boolean
  • Default value:  true
  • Exists by default: yes
  • Application support:Firefox 1.0
  • Status: Active
  • Introduction:
  • Bugs: {{ bug("52154") }} 

Values

true (default)
Syntax hightlighting is enabled.
false
Syntax hightlighting is disabled.
