Represents read-only text, such as labels for other controls or instructions in a dialog box. Static text cannot be modified or selected.

Mapped to

  • ATK: ATK_ROLE_UNKNOWN
  • UA: NSAccessibilityStaticTextRole
  • MSAA/IA2: ROLE_SYSTEM_STATICTEXT

States

  • STATE_READONLY

Used by

  • ARIA: label, description
  • HTML: <label>
  • XUL: <label/>, <description/>
