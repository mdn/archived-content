In Thunderbird this is implemented for different windows in several different places:

  1. nsMsgDBViewCommandUpdater (for the standalone message window)
  2. nsMsgDBViewCommandUpdater (for the threadpane message window)
  3. nsMsgSearchCommandUpdater (for search dialogs)
