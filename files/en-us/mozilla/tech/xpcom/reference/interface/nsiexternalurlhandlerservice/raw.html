Methods

getURLHandlerInfoFromOS()

Given a URL, looks up the handler info from the operating system. This should be overridden by each operating systems implementation.

  in nsIURI aURL,
  out boolean aFound

Parameters
aURL
The URL we are looking for.
aFound
