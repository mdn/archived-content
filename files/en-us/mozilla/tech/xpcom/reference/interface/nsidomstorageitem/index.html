dom/interfaces/storage/nsIDOMStorageItem.idlScriptable
This interface represents an object in DOM storage. This is for session storage.
1.0
66
Introduced
Gecko 1.8
Inherits from: nsISupports Last changed in Gecko 1.8 (Firefox 1.5 / Thunderbird 1.5 / SeaMonkey 1.0)

Gecko 1.9.1 note
Starting in Gecko 1.9.1 (Firefox 3.5), this is only used for session storage; previously, it was also used for global storage.

Attributes

Attribute Type Description
secure boolean

If true, the item was stored for an HTTPS page.

Note:Â All items, regardless of whether they were stored for an HTTP page or an HTTPS page, are visible from HTTPS pages. However, HTTP pages can only see items stored from HTTP pages, and will not see items stored from HTTPS pages. As such, this attribute is irrelevant for storage that is bound to an origin.
value DOMString The value associated with the item.
