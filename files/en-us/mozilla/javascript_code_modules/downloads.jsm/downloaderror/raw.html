A DownloadError object provides detailed information about a download failure.

Properties

Attribute Type Description
result {{ReadOnlyInline()}} nsresult The result code associated with this error.
becauseSourceFailed {{ReadOnlyInline()}} boolean Indicates an error occurred while reading from the remote location.
becauseTargetFailed {{ReadOnlyInline()}} boolean Indicates an error occurred while writing to the local target.

Inherits properties from Error.
