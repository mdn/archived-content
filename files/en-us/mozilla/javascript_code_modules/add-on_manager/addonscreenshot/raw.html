Support for extensions using XUL/XPCOM or the Add-on SDK was removed in Firefox 57, released November 2017. As there is no supported version of Firefox enabling these technologies, this page will be removed by December 2020.

{{LegacyAddonsNotice}}

A screenshot object for an add-on can have following attributes.

Attributes

Attribute Type Description
url string The URL of the screenshot.
thumbnailURL {{optional_inline}} string The URL of a thumbnail version of the screenshot.
caption {{optional_inline}} string The caption of the screenshot.
width {{optional_inline}} string The width of the screenshot.
height {{optional_inline}} string The height of the screenshot.
thumbnailWidth {{optional_inline}} string the thumbnailWidth of the screenshot.
thumbnailHeight {{optional_inline}} string The thumbnailHeight of the screenshot.
