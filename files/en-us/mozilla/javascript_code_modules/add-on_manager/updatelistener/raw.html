An UpdateListener receives messages from an update check for a single add-on, though it is possible to pass the same UpdateListener to as many calls to findUpdates() as you like. For each individual update check, the following methods will be called on the listener:

  • Either onCompatibilityUpdateAvailable() or onNoCompatibilityUpdateAvailable(), depending on whether compatibility information for the requested application version was seen.
  • Either onUpdateAvailable() or onNoUpdateAvailable(), depending on whether a newer version of the Addon was found.
  • onUpdateFinished() to signal that the update check has completed.
