Firefox 3 offers a number of improvements to its CSS implementation. While these changes are documented on the appropriate pages in the documentation, this article provides an overview of them all so that developers can easily learn what these changes were.

  • The {{ Cssxref("display") }} property's inline-block and inline-table values are now implemented.
  • The {{ Cssxref("font-size-adjust") }} property now works on all platforms; previously it was only supported on Windows.
  • rgba() and hsla() support for {{ Cssxref("color") }} ({{ Bug(147017) }})
  • {{ Cssxref(":default") }} pseudo-class support ({{ Bug(302186) }})
  • -moz-max-content, -moz-min-content, -moz-fit-content, and -moz-available values have been added to {{ Cssxref("width") }}, {{ Cssxref("min-width") }}, and {{ Cssxref("max-width") }} ({{ Bug(311415) }} and {{ Bug(402706) }})
  • HTML soft hyphens (&shy;) are now supported.
  • Pre-formatted tabs work much better in proportional fonts, following the CSS 2.1 spec.
  • The {{ Cssxref("ime-mode") }} property is supported.
  • The CSS text-rendering property for HTML is now supported ({{ Bug(387969) }}).
  • -moz-border-*-start and -moz-border-*-end CSS properties are implemented ({{ Bug(74880) }}).
  • -moz-initial is now implemented for almost all CSS properties (except quotes and -moz-border-*-colors) {{ Bug(80887) }}.
  • window.getComputedStyle() now supports all supported CSS properties ({{ Bug(316981) }}).
  • {{ Cssxref("content") }}'s none value is now supported ({{ Bug(378535) }}).
  • {{ Cssxref("cursor") }}'s none value is now supported ({{ Bug(346690) }}).
  • Background images now properly clip to a -moz-border-radius ({{ Bug(24998) }}).
  • {{ Cssxref("white-space") }}'s pre-wrap value is now supported ({{ Bug(261081) }}).
  • Selectors like :first-child, :only-child, :last-child ({{ Bug(73586) }}), :empty ({{ Bug(98997) }}), or the + combinator ({{ Bug(229915) }}) are now updated dynamically ({{ Bug(401291) }}).
  • Negative {{ Cssxref("z-index") }} values are supported correctly.
  • The SVG property {{ Cssxref("text-rendering") }} is now supported and applies to XML and HTML content also ({{ bug("387969") }}).
