You can build the Necko networking engine by itself quite simply, by specifying netwerk as the value of the --enable-application compile option.

Note: Support for building Necko standalone was removed in {{Gecko("9.0")}}.

An example .mozconfig file

ac_add_options --enable-application=netwerk
