Rhino 1.6R1 through 1.6R6 implement E4X using XMLBeans library. If you would like to use E4X you need to add xbean.jar from XMLBeans distribution to your class path. In Rhino 1.6R6 and later the E4X support has been rewritten to rely solely on the DOM3 APIs supported natively by Java 1.5. (Pre-Java 1.5 users can use DOM3 using Java's endorsed standards override mechanism if they have a DOM3-capable XML parser.) If neither XMLBeans nor DOM3 are present, E4X is not available.
