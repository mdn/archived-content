Convert a C floating-point number of type float to a {{jsapixref("JS::Value")}}.

Syntax

JS::Value
JS::Float32Value(float f)
Name Type Description
f float C float to convert.

Description

JS::Float32Value converts a C floating-point number of type float to {{jsapixref("JS::Value")}}, the type of JavaScript values.

The result is an floating-point JS::Value.
