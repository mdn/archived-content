Note: the preferred way of changing a context's current compartment is using {{jsapixref("JSAutoCompartment")}}.

Leave a the compartment, returning to the compartment active before the corresponding JS_EnterCompartment.

Syntax

void
JS_LeaveCompartment(JSContext *cx, JSCompartment *oldCompartment);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context in which to leave the compartment.
oldCompartment {{jsapixref("JSCompartment", "JSCompartment *")}} Value returned by previous call to {{jsapixref("JS_EnterCompartment")}}.

Description

Every {{jsapixref("JSRuntime", "JSContext")}} has a current compartment. Any access to an object in a different compartment must be bracketed by calls to {{jsapixref("JS_EnterCompartment")}} and JS_LeaveCompartment.
