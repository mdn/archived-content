Create a new function based on the given JSFunctionSpec.

Syntax

JSFunction*
JS::NewFunctionFromSpec(JSContext* cx, const JSFunctionSpec* fs, HandleId id);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context in which to define functions. {{ Jsapi-requires-request() }}
fs {{jsapixref("JSFunctionSpec", "const JSFunctionSpec *")}} A pointer to function specification.
behavior {{jsapixref("jsid", "HandleId")}} The result of a successful call to PropertySpecNameToPermanentId(cx, fs->name, &id).

Description

JS::NewFunctionFromSpec creates a new function based on the given JSFunctionSpec, *fs. id is the result of a successful call to PropertySpecNameToPermanentId(cx, fs->name, &id).

Unlike {{jsapixref("JS_DefineFunctions")}}, this does not treat fs as an array. *fs must not be JS_FS_END.

On success, JS::NewFunctionFromSpec returns a pointer to newly created function. On error, it returns nullptr.
