Convert a {{jsapixref("JSObject")}} or null pointer to a {{jsapixref("JS::Value")}}.

Syntax

JS::Value
JS::ObjectOrNullValue(JSObject* obj)
Name Type Description
str {{jsapixref("JSObject", "JSObject*")}} A pointer to a JSObject or null to convert.

Description

JS::ObjectValue converts a given {{jsapixref("JSObject")}} to {{jsapixref("JS::Value")}}. If the pointer is null, it returns JavaScript null. The conversion always succeeds.
