Determines if a given {{jsapixref("jsval")}} is a JavaScript boolean.

Syntax

JSVAL_IS_BOOLEAN(v)

Description

JSVAL_IS_BOOLEAN(v) is true if the given JavaScript value, v, is a boolean value (that is, it is either {{jsapixref("JSVAL_TRUE")}} or {{jsapixref("JSVAL_TRUE", "JSVAL_FALSE")}}).

To convert between JavaScript boolean values (jsval) and C boolean values, use {{jsapixref("JSVAL_TO_BOOLEAN")}} and {{jsapixref("BOOLEAN_TO_JSVAL")}}.
