Note: the preferred way of changing a context's current compartment is using {{jsapixref("JSAutoCompartment")}}.

Enter a different compartment on the given context, so that objects in that compartment can be accessed.

Syntax

JSCompartment *
JS_EnterCompartment(JSContext *cx, JSObject *target);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context on which a cross-compartment call is needed.
target {{jsapixref("JSObject", "JSObject *")}}

The object in a different compartment to be accessed. This implicitly identifies the compartment to be entered.

Description

Every {{jsapixref("JSRuntime", "JSContext")}} has a current compartment. Any access to an object in a different compartment must be bracketed by calling JS_EnterCompartment and {{jsapixref("JS_LeaveCompartment")}}.

On success, JS_EnterCompartment returns a pointer to the previously entered compartment. To return to that compartment, pass it to JS_LeaveCompartment. JS_EnterCompartment is infallible, so a NULL return value doesn't indicate failure.
