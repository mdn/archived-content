Returns the empty string as a {{jsapixref("JSString")}} object.

Syntax

JSString *
JS_GetEmptyString(JSRuntime *rt);
Name Type Description
rt {{jsapixref("JSRuntime", "JSRuntime *")}} The runtime for which to return the empty string.

Description

The returned empty string has the length 0.
