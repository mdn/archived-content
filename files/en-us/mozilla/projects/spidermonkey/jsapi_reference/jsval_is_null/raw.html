Determine if a given {{jsapixref("jsval")}} is null.

Syntax

JSVAL_IS_NULL(v)

Description

JSVAL_IS_NULL(v) is true if v is {{jsapixref("JSVAL_NULL")}}, which is the JavaScript null value.

(Note: {{jsapixref("JSVAL_IS_OBJECT")}}({{jsapixref("JSVAL_NULL")}}) is also true.)

Example

The following code snippet illustrates how a JavaScript variable, MyItem, is conditionally tested in an if statement to see if it contains a null value.

if (JSVAL_IS_NULL(MyItem)) {
    ...
}
