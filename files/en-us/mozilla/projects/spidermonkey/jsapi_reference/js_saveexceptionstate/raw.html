Saves the exception state from the specified context.

Syntax

JSExceptionState *
JS_SaveExceptionState(JSContext *cx);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}

Description

Saves the current exception state (that is, any {{jsapixref("JS_SetPendingException", "pending exception", "", 1)}}, or a cleared exception state) associated with the specified context cx, and returns a {{jsapixref("JSExceptionState")}} object holding this state. That object may then be used to restore again the context using {{jsapixref("JS_RestoreExceptionState")}}, or discarded using {{jsapixref("JS_DropExceptionState")}}. Either of those two functions frees any memory used by the JSExceptionState.

The JSExceptionState created by this function automatically roots any pending exception object which needs to be protected from the GC.

This function returns NULL if the engine was not built with support for exceptions.
