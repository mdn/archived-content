Convert a JS ID to a {{jsapixref("JSProtoKey")}}.

Syntax

JSProtoKey
JS_IdToProtoKey(JSContext *cx, JS::HandleId id);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}
id {{jsapixref("jsid", "JS::HandleId")}} The property identifier to convert.

Description

JS_IdToProtoKey converts a specified JS ID, id, to a prototype key. On success, JS_IdToProtoKey returns a {{jsapixref("JSProtoKey")}}. Otherwise it returns {{jsapixref("JSProto_Null")}}.
