The {{jsapixref("JSObjectOps", "JSObjectOps.defineProperty")}} callback is called whenever an object property is defined. It implements {{jsapixref("JS_DefineProperty")}}.

Syntax

JSBool (*JSDefinePropOp)(JSContext *cx, JSObject *obj, jsid id, jsval value,
    JSPropertyOp getter, JSPropertyOp setter, unsigned int attrs);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to the JS context in which the property is being defined.
obj {{jsapixref("JSObject", "JSObject *")}} The object on which the property is being defined.
id {{jsapixref("jsid")}} The name or index of the property to define.
value {{jsapixref("Jsval", "jsval")}} The initial value for the new property.
getter {{jsapixref("JSPropertyOp")}} The getter for the new property.
setter {{jsapixref("JSPropertyOp")}} The setter for the new property.
attrs {{jsapixref("jsint", "unsigned int")}} The {{jsapixref("JS_GetPropertyAttributes", "property attributes", "#Property_attributes", 1)}} for the new property.

Description

Define obj[id], an own property of obj named id, having the given initial value, with the specified getter, setter, and attributes.
