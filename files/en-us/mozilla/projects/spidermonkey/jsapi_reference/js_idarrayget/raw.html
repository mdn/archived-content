Get the item of a {{jsapixref("JSIdArray")}}.

Syntax

jsid
JS_IdArrayGet(JSContext *cx, JSIdArray *ida, unsigned index)
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} A context.
ida {{jsapixref("JSIdArray", "JSIdArray *")}} The JSIdArray to get the length.
index unsigned The index of the id in the JSIdArray.

Description

JS_IdArrayGet gets the item in the specified index of the ID array pointed to by ida.
