Retrieves a {{jsapixref("JSScript")}} for a specified function.

Syntax

JSScript *
JS_GetFunctionScript(JSContext *cx, JS::HandleFunction fun);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}
fun {{jsapixref("JSFunction", "JS::HandleFunction")}} Pointer to a JS function.

Description

JS_GetFunctionScript returns a pointer to {{jsapixref("JSScript")}} for the specified function, fun. If fun is a native function, it returns NULL.
