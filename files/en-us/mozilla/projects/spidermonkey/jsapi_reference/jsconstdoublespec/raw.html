Describes a double and integer value and assigns it a name.

Syntax

template<typename T>
struct JSConstScalarSpec {
    const char *name;
    T val;
/*
    uint8_t         flags;    // Obsolete from JSAPI 35
    uint8_t         spare[3]; // Obsolete from JSAPI 35
*/
};

typedef JSConstScalarSpec<double> JSConstDoubleSpec;
typedef JSConstScalarSpec<int32_t> JSConstIntegerSpec; // Added in SpiderMonkey 38
Name Type Description
val double or int32_t Value for the double or integer.
name const char * Name to assign the double.
flags uint8_t {{jsapixref("JS_GetPropertyAttributes", "Property attributes", "#Property_attributes", 1)}} for the double. {{ obsolete_inline("jsapi35") }}
Currently these can be 0 or more of the following values OR'd:
  • {{jsapixref("JS_GetPropertyAttributes", "JSPROP_ENUMERATE")}}: property is visible in for loops.
  • {{jsapixref("JS_GetPropertyAttributes", "JSPROP_READONLY")}}: property is read-only.
  • {{jsapixref("JS_GetPropertyAttributes", "JSPROP_PERMANENT")}}: property cannot be deleted.
  • {{jsapixref("JS_GetPropertyAttributes", "JSPROP_EXPORTED")}}: property can be exported outside its object.
  • {{jsapixref("JS_GetPropertyAttributes", "JSPROP_INDEX")}}: property is actually an array element.

If this field is 0, {{jsapixref("JS_DefineConstDoubles")}} uses the default attributes JSPROP_READONLY | JSPROP_PERMANENT instead.
spare uint8_t [3] Reserved for future use. {{ obsolete_inline("jsapi35") }}

Description

JSConstDoubleSpecs is used to define a set of double values that are assigned as properties to an object using {{jsapixref("JS_DefineConstDoubles")}}.

JSConstIntegerSpecs is used to define a set of double values that are assigned as properties to an object using {{jsapixref("JS_DefineConstIntegers")}}.
