JSEnumerateOp is the type of the {{jsapixref("JSClass", "JSClass.enumerate")}} callback.

Syntax

typedef bool
(* JSEnumerateOp)(JSContext *cx, JS::HandleObject obj);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to the JS context in which the enumeration is taking place.
obj {{jsapixref("JSObject", "JS::HandleObject")}} Pointer to the object to be enumerated.

Description

JSEnumerateOp is called just before an object is enumerated (via a for...in statement, an array comprehension, or a call to {{jsapixref("JS_Enumerate")}}). It should define any remaining lazy properties that should be enumerable but are not yet defined in obj. This hook does not implement iteration: once the properties are defined, the JavaScript engine can enumerate them.

JSClass hooks

JSClass offers following hook:

  • The {{jsapixref("JSClass", "JSClass.enumerate")}} hook is for classes that implement lazy properties using {{jsapixref("JSClass.resolve")}}.
