Toggle specified options on a JSContext.

Syntax

uint32
JS_ToggleOptions(JSContext *cx, uint32 options);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} A context on which to modify options.
options uint32 The set of options to toggle. This is the logical OR of zero or more flags described at {{jsapixref("JS_SetOptions")}}.

Description

JS_ToggleOptions toggles context-wide options. It is equivalent to {{jsapixref("JS_SetOptions")}}(cx, {{jsapixref("JS_GetOptions")}}(cx) ^ options).
