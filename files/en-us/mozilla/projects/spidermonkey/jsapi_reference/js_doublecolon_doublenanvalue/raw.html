Create a not-a-number (NaN) floating-point number as a value of type {{jsapixref("JS::Value")}}.

Syntax

JS::Value
JS::DoubleNaNValue()

Description

JS::DoubleNaNValue returns a value of type {{jsapixref("JS::Value")}} that represents an IEEE floating-point quiet Not-a-Number (NaN).

The result is an floating-point JS::Value.
