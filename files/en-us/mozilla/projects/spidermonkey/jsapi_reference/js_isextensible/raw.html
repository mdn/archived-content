Query [[Extensible]] properly of the given object.

Syntax

bool
JS_IsExtensible(JSContext *cx, JS::HandleObject obj, bool *extensible);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context. {{ Jsapi-requires-request() }}
obj {{jsapixref("JSObject", "JS::HandleObject")}} The object to examine.
extensible bool * Out parameter. Receives true if extensible, false otherwise.

Description

JS_IsExtensible queries [[Extensible]] property of the specified object, obj.

If successful, JS_IsExtensible stores [[Extensible]] property to *extensible and returns true, otherwise returns false and the value of *extensible is undefined.
