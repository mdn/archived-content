Free a {{jsapixref("JSIdArray")}}.

Syntax

void
JS_DestroyIdArray(JSContext *cx, JSIdArray *ida);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} A context.
ida {{jsapixref("JSIdArray", "JSIdArray *")}} The JSIdArray to free.

Description

JS_DestroyIdArray frees the ID array pointed to by ida.

ID arrays are primarily the result of {{jsapixref("JS_Enumerate")}}.
