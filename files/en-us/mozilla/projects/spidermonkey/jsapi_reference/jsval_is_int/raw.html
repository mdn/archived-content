Determine if a given {{jsapixref("jsval")}} is a JavaScript number represented in memory as an integer.

Syntax

JSVAL_IS_INT(v)

Description

JSVAL_IS_INT(v) is true if the {{jsapixref("jsval")}} v is a number represented in memory as an integer.

To test whether a value is a number, regardless of how it is represented in memory, use {{jsapixref("JSVAL_IS_NUMBER")}} instead.

To convert between C integer types and jsval, use {{jsapixref("JSVAL_TO_INT")}}, {{jsapixref("INT_FITS_IN_JSVAL")}}, and {{jsapixref("INT_TO_JSVAL")}}.

Example

The following code snippet illustrates how a JavaScript variable, MyItem, is conditionally tested in an if statement to see if it is a JS integer data type.

if (JSVAL_IS_INT(MyItem)) {
    . . .
}
