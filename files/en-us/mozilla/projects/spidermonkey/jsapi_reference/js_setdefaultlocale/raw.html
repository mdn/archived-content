Set and reset the default locale for the {{jsxref("Global_Objects/Intl", "ECMAScript Internationalization API", "", 1)}}.

Syntax

bool
JS_SetDefaultLocale(JSRuntime *rt, const char *locale);

void
JS_ResetDefaultLocale(JSRuntime *rt);
Name Type Description
rt {{jsapixref("JSRuntime")}} * Pointer to a JS runtime
locale const char * String represents locale.

Description

JS_SetDefaultLocale sets the default locale for the {{jsxref("Global_Objects/Intl", "ECMAScript Internationalization API", "", 1)}} ({{jsxref("Global_Objects/Collator", "Intl.Collator")}}, {{jsxref("Global_Objects/NumberFormat", "Intl.NumberFormat")}}, {{jsxref("Global_Objects/DateTimeFormat", "Intl.DateTimeFormat")}}).

Note that the Internationalization API encourages clients to specify their own locales.

The locale string remains owned by the caller.

JS_ResetDefaultLocale resets the default locale to OS defaults.
