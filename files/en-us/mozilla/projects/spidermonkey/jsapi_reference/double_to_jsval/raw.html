Casts a pointer to a GC-allocated jsdouble to type {{jsapixref("jsval")}}.

Please use {{jsapixref("JS::DoubleValue")}} instead in SpiderMonkey 45 or later.

Syntax

jsval
DOUBLE_TO_JSVAL(double d);
Name Type Description
d double C double to convert to a jsval.

Description

DOUBLE_TO_JSVAL is the inverse of {{jsapixref("JS::ToNumber")}}.

To convert a C float, double to type {{jsapixref("JS::Value")}}, use {{jsapixref("JS_NumberValue")}} instead.
