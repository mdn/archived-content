Make a string to interned string.

Syntax

JSString *
JS_InternJSString(JSContext *cx, JS::HandleString str);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context. {{ Jsapi-requires-request }}
str {{jsapixref("JSString", "JS::HandleString")}} A string to be interned.

Description

JS_InternJSString converts a string str to interned string (interned atom) and returns the result string as {{jsapixref("JSString", "JSString *")}}.

On failure, JS_InternJSString returns nullptr.
