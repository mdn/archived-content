Root an internal fixed-size array of {{jsapixref("JS::Value")}}s.

Syntax

JS::AutoValueArray<N> vp(cx);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context in which to add the root. {{ Jsapi-requires-request() }}
N size_t Size of a {{jsapixref("JS::Value")}} array.

Description

JS::AutoValueArray<N> holds a rooted array of {{jsapixref("JS::Value")}}. This is typically used for local variables being passed to function which requires {{jsapixref("JS::HandleValueArray")}} or a pointer to {{jsapixref("JS::Value")}} array.

Examples

Pass to function as a pointer to {{jsapixref("JS::Value")}}.

bool
someFunction1(JSContext *cx, unsigned argc, const JS::Value *argv) {
  /* ... */
}

JS::AutoValueArray<2> args(cx);
/* ... Initialize args[0] and args[1] here ... */
someFunction1(cx, 2, args.begin());

Pass to function as {{jsapixref("JS::HandleValueArray")}}.

bool
someFunction2(JSContext *cx, const JS::HandleValueArray &args) {
  /* ... */
}

JS::AutoValueArray<2> args(cx);
/* ... Initialize args[0] and args[1] here ... */
someFunction2(cx, args);
