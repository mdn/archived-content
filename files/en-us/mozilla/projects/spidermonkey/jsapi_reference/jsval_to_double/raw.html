Casts a given {{jsapixref("jsval")}} to a {{jsapixref("jsdouble")}} without any type checking or error handling.

Syntax

jsdouble
JSVAL_TO_DOUBLE(jsval v);

Description

JSVAL_TO_DOUBLE casts a specified JS value, v, to a C floating-point number of type jsdouble. As a precondition, {{jsapixref("JSVAL_IS_DOUBLE")}}(v) must be true. This function is not type-safe:Â if v is an integer, string, null, or anything else, the behavior is undefined.

To convert any value to a number, use {{jsapixref("JS_ValueToNumber")}} instead.
