Returns the ID of the associated thread for a context.

Syntax

int JS_GetContextThread(JSContext *cx);
Name Type Description
cx JSContext * The context to examine.

Description

JS_GetContextThread returns the ID of the thread currently associated with this context. If the context is not currently associated with any thread, the return value is 0.

See JS_SetContextThread.

JS_GetContextThread is available only if the engine is built with JS_THREADSAFE.
