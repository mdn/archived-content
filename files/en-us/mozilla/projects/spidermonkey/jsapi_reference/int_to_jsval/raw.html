Converts a specified integer value to a JS integer value.

Please use {{jsapixref("JS::Int32Value")}} instead in SpiderMonkey 45 or later.

Syntax

jsval
INT_TO_JSVAL(int32_t i);
Name Type Description
i any integer type C integer to convert to a jsval.

Description

INT_TO_JSVAL converts a C integer, i, to a {{jsapixref("jsval")}}.

Before SpiderMonkey 1.8.5, not all integers can be stored in a jsval; use {{jsapixref("INT_FITS_IN_JSVAL")}} to test. Starting in SpiderMonkey 1.8.5, jsval can store a full 32-bit integer, so this check isn't needed any longer for 32-bit integers.") }}

If i does not fit into a jsval (see {{jsapixref("INT_FITS_IN_JSVAL")}}), the behavior is undefined. To convert any number to a jsval, regardless of whether it fits in an int jsval, use {{jsapixref("JS_NewNumberValue")}} instead.

To convert an integer jsval to a C integer, use {{jsapixref("JS::ToInt32")}}.
