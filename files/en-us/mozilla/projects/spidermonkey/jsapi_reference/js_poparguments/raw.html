Frees the stack entry allocated by {{jsapixref("JS_PushArguments")}}.

Syntax

void
JS_PopArguments(JSContext *cx, void *mark);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}
mark void * Pointer to a void * which holds the stack frame pointer previously supplied by {{jsapixref("JS_PushArguments")}}.

Description

JS_PopArguments frees the stack frame pointer previously allocated by {{jsapixref("JS_PushArguments")}} and unroots the {{jsapixref("jsval")}}s which have been associated with it (those returned by JS_PushArguments as well).
