Determines if a given {{jsapixref("jsval")}} is a number represented as a {{jsapixref("jsdouble")}}.

Syntax

JSVAL_IS_DOUBLE(v)

Description

JSVAL_IS_DOUBLE(v) is true if v is a number represented in memory as a {{jsapixref("jsdouble")}}.

To check whether v is a number, regardless of implementation details, use {{jsapixref("JSVAL_IS_NUMBER")}} instead.

Example

The following code snippet illustrates how a JavaScript variable, MyItem, is conditionally tested in an if statement to see if it is a JS double data type.

if (JSVAL_IS_DOUBLE(MyItem)) {
    . . .
}
