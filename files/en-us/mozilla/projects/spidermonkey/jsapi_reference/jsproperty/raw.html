Internal use only. The type of JavaScript object properties, used by the {{jsapixref("JSObjectOps")}} layer.

Syntax

struct JSProperty {
    jsid id;
};

Description

JSProperty is the abstract base class of all object properties. It is used internally in {{jsapixref("JSObjectOps.lookupProperty")}}, {{jsapixref("JSObjectOps.getAttributes", "getAttributes")}}, {{jsapixref("JSObjectOps.getAttributes", "setAttributes")}}, and {{jsapixref("JSObjectOps.dropProperty", "dropProperty")}}.
