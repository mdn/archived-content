Unregister a pointer to a GC thing so that it is no longer a member of the garbage collector's root set. This enables any pointed-at GC thing to be garbage collected as needed.

Syntax

void
RemoveValueRoot(JSContext *cx, JS::Heap<JS::Value> *vp);

void
RemoveStringRoot(JSContext *cx, JS::Heap<JSString *> *rp);

void
RemoveObjectRoot(JSContext *cx, JS::Heap<JSObject *> *rp);

void
RemoveScriptRoot(JSContext *cx, JS::Heap<JSScript *> *rp);

void
RemoveValueRootRT(JSRuntime *rt, JS::Heap<JS::Value> *vp);

void
RemoveStringRootRT(JSRuntime *rt, JS::Heap<JSString *> *rp);

void
RemoveObjectRootRT(JSRuntime *rt, JS::Heap<JSObject *> *rp);

void
RemoveScriptRootRT(JSRuntime *rt, JS::Heap<JSScript *> *rp);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context from which to remove the root. {{ Jsapi-requires-request() }}
rt {{jsapixref("JSRuntime", "JSRuntime *")}} The runtime from which to remove the root.
vp {{jsapixref("JS::Value", "JS::Heap<JS::Value>")}} Address of the {{jsapixref("JS::Value")}} variable to remove from the root set. This must have been passed to one of {{jsapixref("JS::Add_Root", "JS::AddValueRoot")}}, {{jsapixref("JS::Add_Root", "JS::AddNamedValueRoot")}} or {{jsapixref("JS::Add_Root", "JS::AddNamedValueRootRT")}}.
rp {{jsapixref("JSString", "JS::Heap<JSString *>")}} Address of the {{jsapixref("JSString", "JSString *")}} variable to remove from the root set. This must have been passed to either {{jsapixref("JS::Add_Root", "JS::AddStringRoot")}} or {{jsapixref("JS::Add_Root", "JS::AddNamedStringRoot")}}.
rp {{jsapixref("JSObject", "JS::Heap<JSObject *>")}} Address of the {{jsapixref("JSObject", "JSObject *")}} variable to remove from the root set. This must have been passed to either {{jsapixref("JS::Add_Root", "JS::AddObjectRoot")}} or {{jsapixref("JS::Add_Root", "JS::AddNamedObjectRoot")}}.
rp {{jsapixref("JSScript", "JS::Heap<JSScript *>")}} Address of the {{jsapixref("JSScript", "JSScript *")}} variable to remove from the root set. This must have been passed to {{jsapixref("JS::Add_Root", "JS::AddNamedScriptRoot")}}.

Description

JS::Remove*Root removes the variable that rp points to from the garbage collector's root set.

The entry for vp/rp is removed from the root set for the {{jsapixref("JSRuntime")}} associated with the context cx.
