JS_LeaveCrossCompartmentCall has been removed in {{bug(786068)}}. Use {{jsapixref("JS_LeaveCompartment")}} instead.

Leave a the compartment, returning to the compartment active before the corresponding JS_EnterCrossCompartmentCall.

Syntax

void
JS_LeaveCrossCompartmentCall(JSCrossCompartmentCall *call);
Name Type Description
call JSCrossCompartmentCall * Value returned by previous call to JS_EnterCrossCompartmentCall.

Description

Every {{jsapixref("JSRuntime", "JSContext")}} has a current compartment. Any access to an object in a different compartment must be bracketed by call {{jsapixref("JS_EnterCrossCompartmentCall")}} and JS_LeaveCrossCompartmentCall.
