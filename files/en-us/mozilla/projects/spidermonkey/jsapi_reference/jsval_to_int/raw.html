Converts an integer {{jsapixref("jsval")}} to a C integer without any type checking or error handling.

Syntax

JSVAL_TO_INT(v)

Description

JSVAL_TO_INT converts a specified integer {{jsapixref("jsval")}}, v, to the corresponding C integer value. If v is any other type of value, the result is unspecified.

Note that although jsval is an integer type, it must be treated as opaque. The jsval encoding of the integer 4, for instance, is not ((jsval)Â 4) but some other pattern of bits.

To convert an integer to jsval, use {{jsapixref("INT_FITS_IN_JSVAL")}} and {{jsapixref("INT_TO_JSVAL")}}.
