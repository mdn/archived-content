Determines if a given {{jsapixref("jsval")}} is the JavaScript value undefined.

Syntax

JSVAL_IS_VOID(v)

Description

JSVAL_IS_VOID(v) is true if v is {{jsapixref("JSVAL_VOID")}}, which represents the JavaScript value undefined.

Example

The following code snippet illustrates how a JavaScript variable, MyItem, is conditionally tested in an if statement to see if it is void.

if (JSVAL_IS_VOID(MyItem)) {
    ...
}
