Bind the given callable to use the given object as this.

Syntax

JSObject*
JS_BindCallable(JSContext *cx, JS::Handle<JSObject*> callable,
                JS::Handle<JSObject*> newThis);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}
callable {{jsapixref("JSObject*>", "JS::Handle<JSObject*>")}} Pointer to the function object to bind new this.
newThis {{jsapixref("JSObject*>", "JS::Handle<JSObject*>")}} Pointer to the new this value for callable.

Description

JS_BindCallable binds the given callable to use the given object as this.

If callable is not callable, will throw and return nullptr.
