Determine if a string's characters are stored as Latin1.

Syntax

bool
JS_StringHasLatin1Chars(JSString *str);
Name Type Description
str {{jsapixref("JSString", "JSString *")}} String to examine.

Description

JS_StringHasLatin1Chars returns true iff the string's characters are stored as Latin1.

String characters are stored as either Latin1Char (8-bit) or char16_t (16-bit). Clients can use JS_StringHasLatin1Chars and can then call either the Latin1* or TwoByte* functions. Some functions like {{jsapixref("JS_CopyStringChars")}} and {{jsapixref("JS_GetStringCharAt")}} accept both Latin1 and TwoByte strings.
