bool
JS::CreateError(JSContext *cx, JSExnType type, HandleString stack,
                HandleString fileName, uint32_t lineNumber, uint32_t columnNumber,
                JSErrorReport *report, HandleString message, MutableHandleValue rval);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}
type {{jsapixref("JSExnType")}} The exception's type.
stack {{jsapixref("JSString", "JS::HandleString")}} or {{jsapixref("JSObject", "JS::HandleObject")}} The value of {{jsxref("Error.prototype.stack")}}.
fileName {{jsapixref("JSString", "JS::HandleString")}} The value of {{jsxref("Error.prototype.fileName")}}.
lineNumber uint32_t The value of {{jsxref("Error.prototype.lineNumber")}}.
columnNumber uint32_t The value of {{jsxref("Error.prototype.columnNumber")}}.
report {{jsapixref("JSErrorReport", "JSErrorReport *")}} A pointer to {{jsapixref("JSErrorReport")}} instance.
message {{jsapixref("JSString", "JS::HandleString")}} The value of {{jsxref("Error.prototype.message")}}.
rval {{jsapixref("JS::Value", "JS::MutableHandleValue")}} Out parameter. Receives the created error object.

Description

JS::CreateError creates an {{jsxref("Error")}} object with specified properties.

If successful, JS::CreateError stores the created error object to *rval and returns true, otherwise returns false and the value of *rval is undefined.
