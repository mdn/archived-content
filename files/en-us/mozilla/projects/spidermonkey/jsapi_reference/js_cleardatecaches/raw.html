Clear the cache of calculated local time from each {{jsxref("Global_Objects/Date", "Date")}} object.

Syntax

JS_ClearDateCaches(JSContext *cx);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JavaScript context from which to derive runtime information. {{ Jsapi-requires-request() }}

Description

JS_ClearDateCaches clears the cache of calculated local time from each Date object. Call to propagate a system timezone change.
