Retrieves a pointer to the JSRuntime.

Syntax

JSRuntime *
JS_GetRuntime(JSContext *cx);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context to query.

Description

JS_GetRuntime retrieves a pointer to the JSRuntime with which a specified JSContext, cx, is associated. Each context is associated with a particular JSRuntime when it is created (see {{jsapixref("JS_NewContext")}}); JS_GetRuntime provides a convenient, programmatic way to look up the association.
