Retrieves the constructor for an object.

Syntax

JSObject *
JS_GetConstructor(JSContext *cx, JS::Handle<JSObject*> proto);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} A context. {{ Jsapi-requires-request() }}
obj {{jsapixref("JSObject", "JS::Handle<JSObject*>")}} The object whose constructor is to be returned.

Description

JS_GetConstructor retrieves the constructor property of a given object, obj. If the result is a Function object, JS_GetConstructor returns a pointer to that object. Otherwise, it reports an error and returns NULL.

To get an object's prototype, use {{jsapixref("JS_GetPrototype")}}. To get an object's parent, use {{jsapixref("JS_GetParent")}}.
