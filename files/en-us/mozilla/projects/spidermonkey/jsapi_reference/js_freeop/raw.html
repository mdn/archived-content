A wrapper for {{jsapixref("js_free")}}.

Syntax

void
JS_freeop(JSFreeOp *fop, void *p);
Name Type Description
fop {{jsapixref("JSFreeOp", "JSFreeOp *")}} A pointer to {{jsapixref("JSFreeOp")}} structure to be used to get the implementation of free.
p void * A pointer to the allocated memory to be freed.

Description

JS_freeop is a wrapper for {{jsapixref("js_free", "js_free(p)")}} that may delay {{jsapixref("js_free", "js_free(p)")}} invocation as a performance optimization as specified by the given {{jsapixref("JSFreeOp")}} instance.
