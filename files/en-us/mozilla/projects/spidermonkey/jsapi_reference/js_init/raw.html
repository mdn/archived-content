Initializes the JS engine so that further operations can be performed.

Syntax

#include "js/Initialization.h" // previously "jsapi.h"

bool
JS_Init(void);

Description

Initialize SpiderMonkey, returning true only if initialization succeeded. Once this method has succeeded, it is safe to call {{jsapixref("JS_NewRuntime")}} and other JSAPI methods.

This method must be called before any other JSAPI method is used on any thread. Once it has been used, it is safe to call any JSAPI method, and it remains safe to do so until {{jsapixref("JS_ShutDown")}} is correctly called.

It is currently not possible to initialize SpiderMonkey multiple times (that is, calling JS_Init, JSAPI methods, then {{jsapixref("JS_ShutDown")}} in that order, then doing so again). This restriction may eventually be lifted.

In the past JS_Init once had the signature {{jsapixref("JSRuntime")}} * JS_Init(uint32_t maxbytes) and was used to create new {{jsapixref("JSRuntime")}} instances. This meaning has been removed; use {{jsapixref("JS_NewRuntime")}} instead.
