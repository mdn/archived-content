Returns a JSVersion value representing the version string.

Syntax

JSVersion
JS_StringToVersion(const char *string);
Name Type Description
string const char * Version string to convert.

Description

JS_StringToVersion attempts to convert the version string to a {{jsapixref("JSVersion")}} value. JS_StringToVersion may return any of the following values:

String Enumeration
"1.0" JSVERSION_1_0 {{ obsolete_inline("jsapi24") }}
"1.1" JSVERSION_1_1 {{ obsolete_inline("jsapi24") }}
"1.2" JSVERSION_1_2 {{ obsolete_inline("jsapi24") }}
"1.3" JSVERSION_1_3 {{ obsolete_inline("jsapi24") }}
"1.4" JSVERSION_1_4 {{ obsolete_inline("jsapi24") }}
"ECMAv3" JSVERSION_ECMA_3
"1.5" JSVERSION_1_5 {{ obsolete_inline("jsapi24") }}
"1.6" JSVERSION_1_6
"1.7" JSVERSION_1_7
"1.8" JSVERSION_1_8
Other JSVERSION_UNKNOWN
