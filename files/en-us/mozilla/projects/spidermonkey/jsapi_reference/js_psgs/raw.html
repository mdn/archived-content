Macros for describing properties, for use with {{jsapixref("JS_DefineProperties")}} and {{jsapixref("JS_InitClass")}}.

Syntax

#define JS_PSG(name, getter, flags) ...
#define JS_PSGS(name, getter, setter, flags) ...
#define JS_SELF_HOSTED_GET(name, getterName, flags) ... // Added in SpiderMonkey 31
#define JS_SELF_HOSTED_GETSET(name, getterName, setterName, flags) ... // Added in SpiderMonkey 31

#define JS_PS_END ...
Name Type Description
name const char * The JavaScript name for the property. (or index, if JSPROP_INDEX is present in flags)
getter {{jsapixref("JSNative")}} Pointer to the C/C++ implementation of the getter function.
setter {{jsapixref("JSNative")}} Pointer to the C/C++ implementation of the setter function.
flags {{jsapixref("jsint", "uint16")}} The bitwise OR of any number of {{jsapixref("Property_attributes", "property attributes", "", 1)}}.
getterName const char * The function's name in Self-Hosted JavaScript code for getter function.
setterName const char * The function's name in Self-Hosted JavaScript code for setter function.

Description

These macros encapsulate the definition of {{jsapixref("JSNative")}}-backed {{jsapixref("JSPropertySpec")}}s, by defining the {{jsapixref("JSFunctionSpec", "JSNativeWrappers")}} for them.
