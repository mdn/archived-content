Determine if the given {{jsapixref("JSPropertySpec", "JSPropertySpec::name")}} or {{jsapixref("JSFunctionSpec", "JSFunctionSpec::name")}} value equals the given {{jsapixref("jsid")}}.

Syntax

bool
JS::PropertySpecNameEqualsId(const char *name, JS::HandleId id);
Name Type Description
name const char * {{jsapixref("JSPropertySpec", "JSPropertySpec::name")}} or {{jsapixref("JSFunctionSpec", "JSFunctionSpec::name")}}.
id {{jsapixref("jsid", "JS::HandleId")}} A {{jsapixref("jsid")}} to compare with.

Description

JS::PropertySpecNameEqualsId determines if the given {{jsapixref("JSPropertySpec", "JSPropertySpec::name")}} or {{jsapixref("JSFunctionSpec", "JSFunctionSpec::name")}} value equals the given {{jsapixref("jsid")}}, and returns true if so.
