The {{jsapixref("JSObjectOps")}}.getAttributes and setAttributes callbacks are called to access the attributes of a property.

Syntax

typedef JSBool (*JSAttributesOp)(JSContext *cx, JSObject *obj, jsid id,
    JSProperty *prop, unsigned int *attrsp);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to the JS context in which the property access is happening.
obj {{jsapixref("JSObject", "JSObject *")}} The object whose property attributes are being accessed.
id {{jsapixref("jsid")}} The name or index of the property whose attributes are being accessed.
prop {{jsapixref("JSProperty", "JSProperty *")}} The property whose attributes are being accessed. This may be NULL.
attrsp unsigned int * In/out parameter. See the Description section below.

Description

Get or set attributes of the property obj[id]. Returns JS_FALSE on error or exception, else JS_TRUE with current attributes in *attrsp. If prop is non-null, it must come from the *propp out parameter of a prior {{jsapixref("JSObjectOps.defineProperty")}} or {{jsapixref("JSObjectOps.lookupProperty")}} call.

For more information on property attributes, see {{jsapixref("JS_GetPropertyAttributes")}}.
