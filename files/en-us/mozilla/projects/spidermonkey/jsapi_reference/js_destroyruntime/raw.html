Frees a JavaScript runtime.

Syntax

void
JS_DestroyRuntime(JSRuntime *rt);
Name Type Description
rt {{jsapixref("JSRuntime", "JSRuntime *")}} The runtime to destroy.

Description

JS_DestroyRuntime frees the specified the JavaScript runtime environment, rt. Call JS_DestroyRuntime after completing all other JS API calls. JS_DestroyRuntime garbage collects and frees the memory previously allocated by {{jsapixref("JS_NewRuntime")}}.
