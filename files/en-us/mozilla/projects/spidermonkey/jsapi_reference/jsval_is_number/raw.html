Determine if a given {{jsapixref("jsval")}} is a JavaScript number.

Syntax

JSVAL_IS_NUMBER(v)

Description

JSVAL_IS_NUMBER(v) is true if v is a JavaScript number.

To convert a numeric jsval to a C floating-point number, use {{jsapixref("JS_ValueToNumber")}}.

Example

The following code snippet illustrates how a JavaScript variable, MyItem, is conditionally tested in an if statement to see if it is a JS integer or double value.

if (JSVAL_IS_NUMBER(MyItem)) {
    ...
}
