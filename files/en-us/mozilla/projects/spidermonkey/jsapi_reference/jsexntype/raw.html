Possible exception types.

Value Prototype in JavaScript
JSEXN_NONE An unthrowable error. (lower bound)
JSEXN_ERR {{jsxref("Error")}}
JSEXN_INTERNALERR {{jsxref("InternalError")}}
JSEXN_EVALERR {{jsxref("EvalError")}}
JSEXN_RANGEERR {{jsxref("RangeError")}}
JSEXN_REFERENCEERR {{jsxref("ReferenceError")}}
JSEXN_SYNTAXERR {{jsxref("SyntaxError")}}
JSEXN_TYPEERR {{jsxref("TypeError")}}
JSEXN_URIERR {{jsxref("URIError")}}
JSEXN_LIMIT (upper bound)

Description

These types are part of a {{jsapixref("JSErrorFormatString")}} structure. They define which error to throw in case of a runtime error. JSEXN_NONE marks an unthrowable error.
