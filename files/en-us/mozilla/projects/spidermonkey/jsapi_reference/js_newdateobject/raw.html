Creates a new {{jsxref("Global_Objects/Date", "Date")}} object instance for the given time and date.

Syntax

JSObject *
JS_NewDateObject(JSContext *cx, int year, int mon, int mday, int hour, int min, int sec);
Name Type Description
cx JSContext * The context in which to create the new {{jsxref("Global_Objects/Date", "Date")}} object. {{ Jsapi-requires-request() }}
year int The year to set the new object to.
mon int The month to set the new object to.
mday int The day of the month to set the new object to.
hour int The hour to set the new object to.
min int The minute to set the new object's time to.
sec int The second to set the new object's time to.

Description

Creates and returns a new {{jsapixref("JSObject")}} representing a JavaScript {{jsxref("Global_Objects/Date", "Date")}} object, which is pre-configured using the specified values.
