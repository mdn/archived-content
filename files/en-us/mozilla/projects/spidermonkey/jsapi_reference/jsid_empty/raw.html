Internal {{jsapixref("jsid")}} value which is used in type inference.

Syntax

const jsid JSID_EMPTY;

const JS::HandleId JSID_EMPTYHANDLE; // Added in SpiderMonkey 31

Description

JSID_EMPTY is an internal {{jsapixref("jsid")}} value which is used in type inference code.

JSID_EMPTYHANDLE is the handle to the {{jsapixref("jsid")}} with the value of JSID_EMPTY.
