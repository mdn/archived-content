Get the characters and the length of a string.

Syntax

const jschar *
JS_GetStringCharsAndLength(JSContext *cx, JSString *str, size_t *length);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context. {{ Jsapi-requires-request() }}
str {{jsapixref("JSString", "JSString *")}} A string to get characters.
length size_t * Out parameter. Receives the length of the string.

Description

JS_GetStringCharsAndLength gets the characters and the length of the string, str

If successful, JS_GetStringCharsAndLength returns a pointer to the string, and store the length to *length, otherwise returns NULL
