Convert a C floating-point number of type double to a {{jsapixref("JS::Value")}}.

Replacement to {{jsapixref("JS_NewNumberValue")}}, {{jsapixref("JS_NewDoubleValue")}}, {{jsapixref("JS_NewDouble")}}.

Syntax

// Added in SpiderMonkey 42
JS::Value
JS_NumberValue(double d);

// Obsolete since SpiderMonkey 42
jsval
JS_NumberValue(double d);
Name Type Description
d double The numeric value to convert.

Description

JS_NumberValue converts a C floating-point number of type double to {{jsapixref("JS::Value")}}, the type of JavaScript values.

The result is an integer JS::Value if d can be stored that way. Otherwise, a new floating-point JS::Value is created.
