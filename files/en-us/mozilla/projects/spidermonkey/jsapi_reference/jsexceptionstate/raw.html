The fields of this structure are private/opaque. This is used to save and restore exception states.

Syntax

struct JSExceptionState;

Description

A JSExceptionState object is returned by the {{jsapixref("JS_SaveExceptionState")}} function, and is passed to functions {{jsapixref("JS_RestoreExceptionState")}} and {{jsapixref("JS_DropExceptionState")}}.
