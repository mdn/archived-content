Retrieves the original, canonical Function.prototype for an object's global object.

Syntax

JSObject *
JS_GetFunctionPrototype(JSContext *cx, JS::HandleObject forObj);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JavaScript context from which to derive runtime information. {{ Jsapi-requires-request() }}
forObj {{jsapixref("JSObject", "JS::HandleObject")}} An object from the global whose Function.prototype is being retrieved.

Description

JS_GetFunctionPrototype() retrieves the original Function.prototype of a specified object, obj.Â  If an error occurs, it returns NULL.

Note: This expression might have different values over time if the global Function property is modified, but this method returns only the original value.
