Syntax

bool
JS_DefineOwnProperty(JSContext *cx, JS::HandleObject obj, JS::HandleId id,
                     JS::HandleValue descriptor, bool *bp);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context. {{ Jsapi-requires-request() }}
obj {{jsapixref("JSRuntime", "JS::HandleObject")}} Object on which the property should be defined own.
id {{jsapixref("jsid", "JS::HandleId")}} The jsid of the property that should be defined.
descriptor {{jsapixref("JS::Value", "JS::HandleValue")}} This should be an jsval consisting of an object interpretable as property descriptor.
bp bool * true if successful.

Description

JS_DefineOwnProperty implements the ECMAScript defined function {{jsxref("Object.defineProperty")}}. So the same restrictions apply as for that function (e.g. it is not possible to change a non-configurable property).

descriptor is supposed to be a property descriptor, this means you need to create an object with properties such as value, writable, get or set. See Object.defineProperty for a list of possible fields. A getter or setter defined with this functions will be observable from JS code.
