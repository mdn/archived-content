Convert a C signed 32-bit integer of type int32_t to a {{jsapixref("JS::Value")}}.

Syntax

JS::Value
JS::Int32Value(int32_t i32)
Name Type Description
i32 int32_t C integer to convert.

Description

JS::Int32Value converts a C signed 32-bit integer of type int32_t to {{jsapixref("JS::Value")}}, the type of JavaScript values.

The result is an integer JS::Value.
