JSHasInstanceOp is the type of {{jsapixref("JSClass", "JSClass.hasInstance")}}.

Syntax

typedef bool
(* JSHasInstanceOp)(JSContext *cx, JS::HandleObject obj, JS::MutableHandleValue vp,
                    bool *bp);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The JS context in which the type check is occurring.
obj {{jsapixref("JSObject", "JS::HandleObject")}} The "type" in the type check.
v {{jsapixref("JS::Value", "JS::MutableHandleValue")}} The value whose type is being checked.
bp bool * Out parameter. On success, the callback stores the result of the type check in *bp.

Description

JSHasInstanceOp is called to check whether v is an instance of obj. Return false on error or exception, true on success with true in *bp if v is an instance of obj, false in *bp otherwise.

JSClass hooks

JSClass offers the following hook:

  • The {{jsapixref("JSClass", "JSClass.hasInstance")}} callback implements {{jsapixref("JS_HasInstance")}} and the JavaScript instanceof keyword.
