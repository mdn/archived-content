Get the currently enabled {{jsapixref("JSRuntime", "JSContext")}} options.

Syntax

uint32
JS_GetOptions(JSContext *cx);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context from which to read options.

Description

JS_GetOptions retrieves the option flags of a given JS context cx.

This function returns a uint32 value that is the logical OR of zero or more of the JSOPTION flags described in {{jsapixref("JS_SetOptions")}}.
