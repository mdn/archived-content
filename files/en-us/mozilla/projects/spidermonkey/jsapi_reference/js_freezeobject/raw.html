Freeze an object.

Syntax

bool
JS_FreezeObject(JSContext *cx, JS::Handle<JSObject*> obj);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context. {{ Jsapi-requires-request() }}
obj {{jsapixref("JSObject", "JS::Handle<JSObject*>")}} An object to freeze.

Description

Freezes an object. This means that other code cannot delete, add or change any properties on the object. See ES5's {{jsxref("Object.freeze")}} method.
