Casts a specified JS object to a JS value.

Please use {{jsapixref("JS::ObjectValue")}}/{{jsapixref("JS::ObjectOrNullValue")}} instead in SpiderMonkey 45 or later.

Syntax

jsval
OBJECT_TO_JSVAL(JSObject *obj);
Name Type Description
obj {{jsapixref("JSObject", "JSObject *")}} A pointer to a JavaScript object to convert to a jsval.

Description

OBJECT_TO_JSVAL casts obj from type {{jsapixref("JSObject", "JSObject *")}} to {{jsapixref("jsval")}}. If obj is NULL, the result is {{jsapixref("JSVAL_NULL")}}.
