Convert any JavaScript value to an unsigned 64bit integer.

Syntax

bool
JS::ToUint64(JSContext *cx, JS::HandleValue v, uint64_t *out);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} The context in which to perform the conversion. {{ Jsapi-requires-request() }}
v {{jsapixref("JS::Value", "JS::HandleValue")}} The value to convert.
out uint64_t * Out parameter. On success, *out receives the converted floating-point value.

Description

JS::ToInt64 converts a JavaScript value to an unsigned 64bit integer.

On success, JS::ToInt64 stores the converted value in *out and returns true. On error or exception, it returns false, and the value left in *out is undefined.
