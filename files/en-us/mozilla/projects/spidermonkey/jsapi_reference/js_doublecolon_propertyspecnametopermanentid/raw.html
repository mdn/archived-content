Create a {{jsapixref("jsid")}} that does not need to be marked for GC.

Syntax

bool
JS::PropertySpecNameToPermanentId(JSContext *cx, const char *name, jsid *idp);
Name Type Description
cx {{jsapixref("JSRuntime", "JSContext *")}} Pointer to a JS context from which to derive runtime information. {{ Jsapi-requires-request() }}
name const char * {{jsapixref("JSPropertySpec", "JSPropertySpec::name")}} or {{jsapixref("JSFunctionSpec", "JSFunctionSpec::name")}}.
idp {{jsapixref("jsid", "jsid *")}} Out parameter. Receives the created id.

Description

JS::PropertySpecNameToPermanentId creates a {{jsapixref("jsid")}} that does not need to be marked for GC from {{jsapixref("JSPropertySpec", "JSPropertySpec::name")}} or {{jsapixref("JSFunctionSpec", "JSFunctionSpec::name")}}. The resulting {{jsapixref("jsid")}}, on success, is either an interned string or a well-known symbol; either way it is immune to GC so there is no need to visit *idp during GC marking..
