El pseudo-element CSS ::-ms-track representa la "pista" (la ranura en la qual llisca l'indicador) d'un {{HTMLElement("input")}} de type="range".

Especificacions

No forma part de cap especificació.

Microsoft té una descripció en MSDN .

  • Els pseudo-elements utilitzats per l'IE / Edge per donar estil altres parts d'un rang d'entrada

    • {{cssxref("::-ms-fill-upper")}}
    • {{cssxref("::-ms-fill-lower")}}
  • Pseudo-elements similars utilitzats per altres navegadors:
    • {{cssxref("::-webkit-slider-runnable-track")}}
